# ☕ CoffeeBot Assistant – Final Project

## 📌 Project Overview
**CoffeeBot Assistant** is an AI-powered customer service chatbot designed specifically for coffee shop customers.  
This chatbot helps users:
- Choose coffee based on taste preferences
- Understand differences between coffee menu items
- Get personalized drink recommendations
- Learn basic coffee knowledge and brewing insights

---

## 🎯 Use Case
**Customer Service Bot for Coffee Shop Industry**

CoffeeBot acts as a virtual barista that enhances customer experience by providing fast, relevant, and personalized coffee-related assistance.

---

## 🚀 Features
- 🤖 AI-powered chatbot using Google Gemini
- ☕ Coffee-specific knowledge base
- 🎯 Personalized recommendations based on user preferences
- 🧠 Session memory for interactive conversations
- 🎭 Tone customization (Friendly / Formal / Casual)
- 💻 Interactive Streamlit user interface

---

## ▶️ Installation & Run
1. Clone repository
```bash
git clone https://github.com/yourusername/coffee-chatbot-final-project.git
cd coffee-chatbot-final-project
```
2. Install dependencies
```bash
pip install -r requirements.txt
```
3. Run Streamlit
```bash
streamlit run app.py
```

## 🔑 API Key

To use CoffeeBot, enter your Google Gemini API Key in the sidebar input field after launching the app.
